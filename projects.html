Projects

Software development is always challenging but also incredibly rewarding. That flash of brilliance as you suddenly realize the key to the problem is one of the most exhilirating feelings ever. Here, I present some of my school and other miscallaneous projects.

⇩

SynBioCAD (JBEI)

A program written for the Joint BioEnergy Institute with Leo d'Espaux. It determines how to build DNA cassettes to genetically modify yeast for biofuel production purposes. Written in Python. Deployed using Django on an internal server.

GC-MS Analysis Tool (JBEI)

A data analysis tool that reads raw data output by a gas chromatography mass spectrometer and organizes and presents data based on user preferences. Written in Python. Deployed using Django on an internal server.

Trie Applications (Data Structures)

The final project of CS61B. This involved using tries (specifically ternary search tries) to sort words using user defined alphabets and autocomplete user inputs with those same dictionaries. Written in Java.

Gitlet (Data Structures)

A version control system fairly similar to git, but is smaller and simpler. Implemented many of the features of git, such as initialize, add, commit, remove, log, find, status, checkout, branch, merge, etc. Written in Java.

NGordnet (Data Structures)

Program allowing for the querying of the popularity of words over time, the relative popularity of categories of words, the hyponyms of words, as well as the frequency of words of certain lengths. This program gathered data from the NGrams Dataset in order to achieve this. Written in Java.

Checkers61B (Data Structures)

A game of checkers with a twist. Upon capturing with a bomb piece, all adjacent pieces in a 3x3 block kill all non-shield pieces within range. Written in Java.
